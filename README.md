# Computer Vision & Machine Learning 👁️🤖

This repository contains practical and laboratory works from the Digital Image Processing and Machine Learning course (Master's program, Data Engineering, Lviv Polytechnic National University).

The goal of the project is to explore the evolution of approaches in Computer Vision: from basic pixel manipulation and mathematical filters to building Convolutional Neural Networks (CNN) for image classification.

## 🛠 Tech Stack
* **Language:** Python 3.11
* **Image Processing:** OpenCV, Pillow, NumPy
* **Machine Learning:** Scikit-Learn (KNN, SVM, Logistic Regression)
* **Deep Learning:** TensorFlow, Keras
* **Visualization:** Matplotlib, Seaborn

## 🚀 How to Run Locally

To run the project on your machine, execute the following steps in your terminal:

```bash
# 1. Clone the repository and navigate to the folder
git clone https://github.com/Karoshi-man/CV
cd CV

# 2. Create and activate a virtual environment (Python 3.11 recommended)
py -3.11 -m venv .venv
.venv\Scripts\activate

# 3. Install all required dependencies
pip install tensorflow scikit-learn opencv-python numpy matplotlib seaborn jupyter
```
After successful installation, you can open the project in VS Code or run Jupyter Notebook to view and execute the code
